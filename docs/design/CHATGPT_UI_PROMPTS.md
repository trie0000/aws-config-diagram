# ChatGPT 画面イメージ生成プロンプト集

> 目的: 各画面のUIモックアップ画像をChatGPT (DALL-E) で生成する
> 使い方: 各プロンプトをそのままChatGPTに貼り付けて画像生成を依頼する

---

## 共通指示（最初に1回だけ貼る）

```
これから「AWS Config Diagram Generator」というWebアプリのUI画面モックアップを複数枚作成してもらいます。

■ アプリの概要
AWS Config Snapshot JSON をアップロードすると、AWSネットワーク構成図を自動生成するツールです。ブラウザ上で構成図の表示・編集ができ、Excel/PowerPointにエクスポートできます。Figma や draw.io のようなエディタ型UIです。

■ デザインの方向性
- モダンでプロフェッショナルなダークテーマUI
- 参考イメージ: Figma、Linear、Vercel Dashboard、AWS Console のダークモード
- カラーパレット:
  - 背景(メイン): #0f172a (スレートネイビー)
  - 背景(サイド): #1e293b
  - カード/パネル: #334155
  - アクセント: #3b82f6 (ブルー)
  - 成功: #22c55e (グリーン)
  - 警告: #f59e0b (アンバー)
  - 危険: #ef4444 (レッド)
  - テキスト: #f8fafc (ホワイト)
  - テキスト(サブ): #94a3b8 (グレー)
- フォント: Inter / SF Pro Display 系のモダンなサンセリフ
- 角丸: 8px (カード/ボタン), 12px (モーダル)
- シャドウ: 控えめなドロップシャドウ
- AWSリソースは公式アイコンスタイル（オレンジ系のアイコン）
- 解像度: 1920×1080 (フルHDデスクトップ)
- 言語: 日本語 UI（メニュー・ラベル・プレースホルダー全て日本語）
- 画像内にテキストを入れる場合は読める大きさで

以上の条件で統一してください。では1枚目から始めます。
```

## P03: Canvas拡大（構成図の詳細）

```
AWS構成図エディタのCanvas部分を拡大したUIモックアップを生成してください。ツールバーやパネルは不要で、構成図そのものをクローズアップ。

■ 表示する構成図の内容

構成図は以下の4階層ネスト構造で表現:

レイヤー1（最外）: 「AWS Cloud」の薄いグレー枠
レイヤー2: VPC枠（緑ボーダー、角丸、「Production VPC 10.0.0.0/16」ラベル）
レイヤー3: 2つのAZ（「ap-northeast-1a」「ap-northeast-1c」の枠、横並び）
レイヤー4: 各AZ内に3つのSubnet枠:
  - Public Subnet（薄い緑、「10.0.1.0/24」）→ 中にEC2アイコン、NAT Gatewayアイコン
  - Private Subnet（薄い青、「10.0.2.0/24」）→ 中にEC2アイコン×2、ECSアイコン
  - Isolated Subnet（薄い紫、「10.0.3.0/24」）→ 中にRDSアイコン、ElastiCacheアイコン

VPC外の要素:
- VPC左端に: IGWアイコン（Internet Gateway、小さめ）
- VPC外の左側に3つ（縦並び）: Route53、CloudFront、API Gateway
- VPC外の下部に4つ（横並び）: S3、DynamoDB、SQS、Lambda

接続線:
- CloudFront → ALB（VPC内Public Subnet）への矢印線
- ALB → EC2 × 2 への矢印線（分岐）
- EC2 → RDS への矢印線
- NAT Gateway → IGW への矢印線

■ デザイン
- 背景: 薄いドットグリッド（#0f172a 上に #1e293b のドット）
- AWSリソースのアイコン: 各アイコンの下に白文字のラベル
- 接続線: 白またはグレーの実線、矢印ヘッド付き
- 選択中のリソース（EC2 1つ）: 青いハイライト枠 + 四隅にリサイズハンドル
- 各枠のラベル: 左上に配置（枠名 + CIDR）
```
